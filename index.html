Projetos em Ciência de Dados

Construção de modelo de previsão de receitas para a rede de farmácias Rossmann

Nesse projeto, utilizei linguagem de programação Python, técnicas estatísticas, algoritmos de seleção de features, técnicas de pré-processamento e técnicas de machine learning para desenvolver um modelo de previsão de receitas para as próximas seis semanas para a rede de farmácias Rossmann. Ao final, o modelo apresentou desempenho satisfátorio, com a métrica Erro Médio Percentual Absoluto de 11,84% no primeiro ciclo do CRISP-DM.

As ferramentas utilizadas foram:

  • Git, Gitlab e GitHub.
  • Python, Pandas, Matplotlib e Seaborn.
  • Jupyter Notebook.
  • Regressão Linear, Regressão Linear Regularizada, Random Forest, XGBoost Regressor.
  • Flask, Cloud Render.
  • Desenvolvimento de um Bot no Telegram para consulta ao modelo.

Classificação de pacientes para diagnóstico de diabetes

O projeto foi desenvolvido utilizando linguagem de programação R, técnicas estatística e de machine learning para a classificação de pacientes de acordo com a predisposição para o desenvolvimento de diabetes. Utilizando de um conhecido banco de dados público, o objetivo do projeto era apresentar uma solução com acurácia superior a 75%. Ao final, utilizando o algoritmo de classificação SVM com Kernel SIGMA apresentou acurácia de 79,91%

As ferramentas utilizadas foram:

  • R, Dplyr, GGplot2.
  • R Studio e Visual Code Studio
  • Algoritmos de classificação KNN, Naive Bayes, Random Forest, SVM com Kernel Radial, SVM com Kernel Sigma

Classificação de predição de satisfação de clientes

O projeto é o resultado de minha participação na competição Hackday DS - 5ª edição, realizado em julho de 2023, onde fiz parte de um time de 5 cientistas de dados. Durante 24 horas, o objetivo era desenvolver uma solução para a rede de lojas Instyle, que necessitava prever a quais clientes ficariam insatisfeitos com os serviços da loja de acordo com a avaliação dada aos serviços fornecidos. Ao final, desenvolveu-se um modelo utilizando o algoritmo LGBM, resultando em acurácia de 94,17%.

As ferramentas utilizadas foram:

  • Git, Gitlab e GitHub.
  • Python, Pandas, Matplotlib e Seaborn.
  • Jupyter Notebook e Visual Studio Code.
  • SVM, Random Forest, XGBoost, LGBM.
